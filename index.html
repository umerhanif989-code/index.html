Steps For Downloading

  1. Go to the Official Website Visit https://playvalorant.com using your browser.
  2. Click on “Play Free”
  3. This will redirect you to the sign-in or sign-up page for Riot Games.
  4. Sign In or Create a Riot Account
  5. If you already have a Riot account (used for games like League of Legends), sign in.
  6. If not, click "Make One" to create a free account.
  8. Download the Installer After signing in, click the “Download” button to get the Valorant installer (around 70–100 MB).
  9. Run the Installer (Install Valorant) Open the downloaded .exe file and follow the installation prompts. This installs the Riot Client.
  10. Download the Full Game Once Riot Client is installed, it will begin downloading the full Valorant game (~25–30 GB). Wait for the download to complete.
  11. Install Vanguard (Anti-Cheat) Valorant uses Riot’s anti-cheat system, Vanguard, which installs automatically. You may need to restart your PC after this.
  12. Launch Valorant After installation and restart, open the Riot Client and click "Play" to start Valorant.
  13. Log In and Start Playing Sign in again with your Riot account if prompted, and you’re ready to play!

Common Valorant Errors

System Requirements Not Met
Occurs if your PC doesn't meet the minimum system requirements. Check your hardware (CPU, RAM, GPU) and upgrade if needed.
Installer Stuck or Not Opening
Caused by antivirus/firewall blocking the installer. Temporarily disable antivirus or run installer as Administrator.
Vanguard Installation Failed
Vanguard (anti-cheat) sometimes fails to install properly.
